# æro-agents

A unified environment where AI agents work together from a single workspace.

## Vision

Most AI tools solve one problem at a time.

Aero Agents is being built around a different idea:

Multiple specialized agents.
One environment.
One workflow.

The goal is to reduce friction between thinking and execution.

## Current Status

Early development.

Building in public.

## Roadmap

- Core agent framework
- Terminal integration
- Workflow orchestration
- Multi-agent communication
- Knowledge management
- Local and cloud execution

More updates soon.
