# Roadmap

A living document. Order shifts as we learn.

## Phase 1 - Foundation
- Multi-model chat across every frontier provider
- Per-conversation model switching
- Usage-based credits, no subscriptions
- Wallet-based accounts

## Phase 2 - Agents
- User-created agents with custom goals and tools
- Preset agent library
- File-aware sessions (view, edit, diff)

## Phase 3 - Teams
- Multi-agent teams with shared context
- Orchestration and hand-off between agents
- Run history and replay

## Phase 4 - Automation
- Scheduled agents and recurring tasks
- Analyzers (GitHub, YouTube, account legitimacy)
- Notifications and digests

## Phase 5 - Reach
- Developer API and credit packs
- Telegram and beyond
- Public agent sharing
