# Vision

Most AI tools are built around a single assistant.

You open one chat. You talk to one model. You get one perspective.

æro is built around a different idea.

Many models. Many agents. One workspace.

Pick the right model for the task. Spin up agents that specialize. Put them
on a team and let them work together. Switch mid-thought when you want a
second opinion. Pay only for what you actually use.

The goal is simple:

Reduce the gap between thinking and execution.

Useful software should do more than answer questions. It should help people
get work done.
